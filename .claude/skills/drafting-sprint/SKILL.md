# 7단계 · 초안 쓰기 (Drafting Sprint)

## 역할
'편집자 모드'를 끄고 '창작자 모드'로 질주한다. 완벽함을 버리고 백지를 채우는 것에 집중.

## 분기 — 모드 선택

AskUserQuestion으로 묻는다:
> "초안 작성 방식을 선택해주세요. (1) 직접 10분 스프린트 / (2) AI에게 초안 요청"

---

## [모드 1] 직접 쓰기 — 10분 스프린트

### Step 1. 시작 알림
> "지금부터 10분간 초안을 작성하세요. 문법은 신경 쓰지 마세요. 타이머를 시작합니다. ⏱️"
>
> (실제 타이머 기능이 없으면) "10분 후 다시 저를 불러주세요."

### Step 2. 러닝메이트 독려
작가가 중간에 "이 표현 맞나?", "여기서 막혀요" 같은 메시지를 보내면:
- "문법은 나중에 고칩시다. **계속 쓰세요.**"
- "이미 절반 지났습니다. 속도를 유지하세요. 🏃"
- "지금 떠오른 단어 그대로 적으세요. 다듬지 말고."

### Step 3. 10분 종료
> "🎉 초안 완성을 축하합니다! 백지를 정복하셨습니다.
> 이제 8단계(직접 다듬기)로 넘어갈까요?"

---

## [모드 2] AI 초안 요청

### Step 1. 입력 종합
다음을 모두 로드:
- 5단계 목차 (`writing-workspace/05-outline.md` 또는 메모리)
- 6단계 문체 프로필 (`writing-workspace/06-voice-profile.md`)
- 3단계 자료 목록 (작가 인터뷰 에피소드 포함)

### Step 2. 70% 완성도 초안 작성

**원칙:**
- **의도적으로 거칠게** — 작가가 손볼 여지를 남긴다
- 문체 프로필을 충실히 따른다 (어조·문장 길이·은유 패턴)
- 작가의 개인 경험담을 적재적소에 녹인다
- 기술 용어에는 일상 비유를 덧붙여 온도감 부여
- 완벽한 결말 대신 "여기는 작가님 손이 필요합니다"식 빈칸 표시 가능

### Step 3. 산출
```markdown
# [제목 임시]

## 1. [도입부 소제목]
... (300자, 70% 완성도)

## 2. [본론 소제목]
... (1,500자, 일부 [TODO: 작가 손길] 표시)

## 3. [결말 소제목]
... (500자)
```

### Step 4. 안내 메시지
> "70% 수준의 초안입니다. 작가님 색깔로 손볼 여지를 남겼습니다.
> 특히 [TODO] 표시된 곳은 작가님의 경험으로 채워야 살아납니다.
> 8단계(직접 다듬기)로 넘어갈까요?"

## 입력 / 출력

**입력:**
- 5단계 목차 + 6단계 문체 프로필 + 3단계 자료
- 작가의 모드 선택

**출력:**
- (모드 1) 작가 직접 초안 + 격려 메시지
- (모드 2) **70% 완성도 초안 전문**
- 다음 단계(`self-correction`) 진행 여부 확인
