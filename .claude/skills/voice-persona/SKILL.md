# 6단계 · 나의 문체(스타일) 모방하기 (Voice & Persona)

## 역할
작가 고유의 페르소나를 강화한다. 딱딱한 기술 용어는 부드럽게 풀고, 감성적인 문장에는 논리적 뼈대를 심어 '따뜻한 이성'이 느껴지는 문체를 설정한다.

## 행동 절차

### Step 1. 샘플 글 요청
AskUserQuestion으로 묻는다:
> "작가님의 문체를 학습하고 싶습니다. 과거에 쓰신 글 중 가장 만족스러운 글 1편을 공유해 주시겠어요? (파일 경로/내용 직접 붙여넣기 모두 가능)"

**입력값이 없으면 기본값으로 `docs/6.나의글샘플.txt`를 사용하도록 안내.**
> "샘플이 없으시면 기본 샘플(`docs/6.나의글샘플.txt`)로 진행하겠습니다."

### Step 2. 5축 문체 분석
Read로 샘플 글을 읽고 다음 5개 축으로 분석:

1. **어조(Tone)**: 격식체/반말체, 거리감/친밀감 정도
2. **문장 길이**: 평균 어절 수, 단문/장문 비율
3. **은유 패턴**: 주로 사용하는 비유 소재 (건축/음악/자연/스포츠 등)
4. **어휘 수준**: 한자어/외래어/순우리말 비율
5. **리듬감**: 반복 구조, 대구법, 점층법 사용 여부

### Step 3. MBTI 파악 질문 3종
AskUserQuestion으로 글쓰기 성향을 진단:

1. "글을 쓸 때 논리적 설득과 감정적 공감 중 어느 쪽을 더 중시하시나요?"
2. "독자 반응을 예상하며 쓰시나요, 아니면 쓰고 싶은 대로 쓰시나요?"
3. "글의 구조를 먼저 짜시나요, 아니면 쓰면서 구조가 나오나요?"

### Step 4. 문체 프로필 카드 작성

```markdown
# 📇 문체 프로필 카드

| 항목 | 분석 결과 |
|---|---|
| 어조 | 담담한 경어체, 친밀한 거리 |
| 평균 문장 길이 | 18어절 (단문 40% / 장문 60%) |
| 은유 패턴 | 건축·도시 비유 다용 |
| 어휘 수준 | 한자어 30% / 외래어 15% / 순우리말 55% |
| 리듬감 | 3단 점층, 대구법 자주 |
| MBTI 성향 | T-J 우세 (논리·구조 중시) |

## 작가의 시그니처
- "코드는 시간의 압력을 견디는 건축물이다"류의 시간성·물성 비유
- 단호한 단문 → 사색적 장문의 리듬
- 1인칭이지만 거리감 있는 시선
```

### Step 5. 구체적 가이드라인 3개
이후 단계(7~10)에서 일관성을 유지하기 위한 지침:

- "이 부분은 작가님 특유의 담담한 독백 톤으로 가는 것이 좋겠습니다."
- "기술 용어 등장 시 곧바로 건축 비유로 이어 친근감을 살리세요."
- "결론부에서는 짧은 단문 3개로 점층 마무리하는 패턴을 유지합니다."

### Step 6. 저장
가능하면 프로필 카드를 `./writing-workspace/06-voice-profile.md`에 저장해 7~10단계에서 자동 참조하도록 한다.

## 입력 / 출력

**입력:**
- 작가가 제공한 과거 글 1편 (없으면 `docs/6.나의글샘플.txt`)
- MBTI 파악 질문 답변

**출력:**
- **문체 프로필 카드** (5축 분석 + 시그니처)
- **구체적 가이드라인 3개**
- 다음 단계(`drafting-sprint`) 진행 여부 확인
