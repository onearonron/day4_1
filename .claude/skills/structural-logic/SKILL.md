# 4단계 · 정리하기 (Structural Logic)

## 역할
수집된 재료를 늘어놓고 '버릴 것'과 '취할 것'을 결정한다. 모든 소재가 핵심 메시지를 향해 한 방향으로 정렬되도록 논리의 흐름을 잡는다.

## 행동 절차

### Step 1. 그룹화 (Grouping)
유사한 키워드/테마별로 자료를 묶는다.
- 클러스터 라벨 부여 (예: "실패담", "기술 비교", "철학 인용", "통계")

### Step 2. 우선순위 분류

**1순위 — 핵심 논거 (반드시 포함)**
주제를 직접 증명하는 강력한 사례·논거.

**2순위 — 보조 사례 (논거 강화)**
1순위를 풍부하게 만드는 재료.

**3순위 — 장식 요소 (분위기·여유)**
없어도 글이 성립하지만 풍미를 더하는 인용·디테일.

| 순위 | 항목 | 이유 |
|---|---|---|
| 🥇 1 | 2018 레거시 사례 | 주제 직접 증명, 가장 생생함 |
| 🥈 2 | 클린 코드 인용 | 핵심 논거 보강 |
| 🥉 3 | 도자기 비유 | 결말 분위기용 |

### Step 3. 삭제 권유
주제와 거리가 먼 항목을 별도 표로 정리하고, **삭제 이유**를 명시. 작가에게 최종 결정을 맡긴다.

### Step 4. 톤 앤 매너 일관성 점검
다음 체크리스트로 점수 산출 (5점 만점):

- [ ] 기술 용어와 일상어의 비율이 적절한가? (권장 70:30 이내)
- [ ] 문장 길이가 일관된가? (단문/장문 혼합 패턴)
- [ ] 1인칭/3인칭 시점이 일관된가?
- [ ] 격식/반말 어조가 흔들리지 않는가?
- [ ] 비유 소재가 한두 가지 영역에 모여 있는가?

각 항목 1점, 합계로 톤 점수 표기.

## 입력 / 출력

**입력:**
- 3단계의 자료 목록 + 연결 고리

**출력:**
- **우선순위별 자료 배치도** (1/2/3순위 표)
- **삭제 권장 항목 목록** (이유 포함)
- **톤 앤 매너 일관성 점수** (X/5점, 미흡 항목 코멘트)
- 다음 단계(`blueprint-outline`) 진행 여부 확인
